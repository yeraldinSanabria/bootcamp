Categoria Propiedad Descripción Ejemplo
Texto -- Text color Define el color del texto. color: #ff0000;
font-size Especifica el tamaño de la fuente. font-size: 16px;
text-align Especifica cómo se alinea el texto dentro de un elemento. text-align: center;
text-transform Controla la capitalización del texto. text-transform: uppercase;
line-height Establece la altura de la línea. line-height: 1.5;
Fondo -- Background background-color Establece el color de fondo de un elemento. background-color: #00ff00;
background-image Especifica una imagen de fondo para un elemento. background-image: url('image.jpg');
background-position Establece la posición inicial de una imagen de fondo. background-position: center;
background-size Controla el tamaño de la imagen de fondo. background-size: cover;
Modelo de caja -- Box Model margin Define el espacio exterior de un elemento. margin: 10px;
padding Establece el espacio interior de un elemento. padding: 20px;
border Especifica el borde de un elemento. border: 1px solid #000000;
width Define la anchura de un elemento. width: 100px;
height Define la altura de un elemento. height: 200px;
box-sizing Define cómo se calculan las dimensiones de un elemento. box-sizing: border-box;
Posicionamiento -- Positioning position Determina cómo se posiciona un elemento en el documento. position: absolute;
top Establece la posición superior de un elemento posicionado. top: 10px;
left Establece la posición izquierda de un elemento posicionado. left: 20px;
z-index Controla el orden en el que se apilan los elementos. z-index: 100;
float Hace que un elemento flote a la izquierda o derecha. float: left;
clear Especifica si un elemento debe ser movido debajo de los elementos flotantes. clear: both;
Caja flexible -- Flexbox display Controla cómo se muestra un elemento en la página. display: flex;
flex-direction Establece la dirección en la que los elementos secundarios de un contenedor flexible se colocan. flex-direction: row;
justify-content Define cómo se distribuyen los elementos dentro de un contenedor flexible. justify-content: space-between;
align-items Establece la alineación predeterminada de los elementos flexibles en el eje cruzado. align-items: center;
flex-wrap Controla si los elementos secundarios se envuelven en una nueva línea o no. flex-wrap: wrap;
Grilla para organizar la vista -- Grid display Define un contenedor de cuadrícula y activa el contexto de cuadrícula de sus elementos secundarios. display: grid;
grid-template-columns Especifica el número (y el ancho) de columnas en una cuadrícula. grid-template-columns: 1fr 2fr;
grid-template-rows Especifica el número (y la altura) de filas en una cuadrícula. grid-template-rows: 100px auto;
grid-gap Define el espacio entre filas y columnas en la cuadrícula. grid-gap: 10px;
justify-items Alinea los elementos secundarios a lo largo del eje horizontal en la cuadrícula. justify-items: start;
Animaciones -- Animations animation Aplica una o más animaciones a un elemento. animation: slide 2s infinite;
animation-name Especifica el nombre de la animación que debe ser aplicada. animation-name: fadeIn;
animation-duration Especifica la duración de una animación. animation-duration: 3s;
animation-timing-function Define cómo avanza la animación en cada ciclo. animation-timing-function: ease-in-out;
animation-delay Especifica un retraso antes de que comience la animación. animation-delay: 1s;
Transformación -- Transformations transform Aplica una transformación 2D o 3D a un elemento. transform: rotate(45deg);
transform-origin Define el punto de origen para una transformación. transform-origin: center;
perspective Establece una perspectiva 3D para un elemento. perspective: 1000px;
Transición -- Transitions transition Especifica el efecto de transición entre dos estados de un elemento. transition: all 0.3s ease;
transition-property Define las propiedades a las que se aplica la transición. transition-property: opacity;
transition-duration Establece la duración de la transición. transition-duration: 0.5s;
transition-timing-function Especifica la velocidad de la curva de la transición. transition-timing-function: linear;
transition-delay Define un retraso antes de que comience la transición. transition-delay: 0.2s;
Diseño Responsivo -- Responsive Design media-queries Aplica estilos condicionalmente según el tamaño de la pantalla u otras características del dispositivo. @media (max-width: 600px) { body { font-size: 14px; } }
viewport Controla el tamaño y la escala de la ventana gráfica de un documento. <meta name="viewport" content="width=device-width, initial-scale=1.0">
flexbox Ayuda a crear diseños adaptativos y flexibles. display: flex;
grid Permite crear diseños de cuadrícula que se adaptan a diferentes tamaños de pantalla. display: grid;
Otra -- Other opacity Controla la opacidad de un elemento. opacity: 0.5;
cursor Especifica el tipo de cursor que se muestra cuando se pasa el mouse sobre un elemento. cursor: pointer;
overflow Controla cómo se maneja el contenido desbordado de un elemento. overflow: hidden;
visibility Define si un elemento es visible o no. visibility: hidden;
clip Define una región de recorte para un elemento. clip: rect(0, 0, 100px, 100px);
